About this project

This group project it's actually a homework for a Web-Development Course, we - Cristi, Adi and Ale - are trying to complete by the end of the course. It consists in creating a puzzle with numbers and in answering a short quiz.

Numeric puzzle

Simply by clicking numbers around the empty cell, you can move them inside the empty cell, one by one. The idea is to order them from 0 to 14 inside the table.

Quiz

Here we have tried to create a simple quiz related to numbers, a quiz that can be checked by clicking a button.


P.S. In doing this group project we used HTML, CSS, JavaScript, jQuery, PHP, mySQL.
